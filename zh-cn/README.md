# 什么是Kothonva?
***无法表达的语言***

# Kothonva的特点

* **能够直接使用 Py 运行**
* **运行速度快，编译速度快**
* **轻巧、占用空间小或内存小**
* **语法简洁明了**
* **它的使用更像是一种语言，而不是超集**
